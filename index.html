Help Menu

Command Options

SPD - Adjusts speed value displayed in flight marker datablock.

*Should be entered as an integer value between 0 and 999.

ALT - Adjusts altitude value displayed in flight marker datablock.

*Should be entered as an integer value between 0 and 999.

HDG - Adjusts heading value dislpayed in flight marker datablock. Reorients flight marker.

*Should be entered as an integer value betewen 0 and 360 inclusive.

Example

Commands should be entered in the following format: CALLSIGN COMMAND VALUE

To change the heading of flight with call sign N7601S to a bearing of 105 degrees:

N7601S HDG 105

Type command into command box at bottom left of screen and click submit or press enter to execute command

On Screen Keyboard

The keyboard button can be used to display an on screen keyboard for command input

Toggle Airport Markers

The second button in the bottom control bar (in between the keyboard and pause button) is used to toggle the local airport markers on and off the screen. Visibility can be adjusted before, during and after simulation

Pause and Resume

The pause button will freeze all flight markers on the screen until the resume button is clicked.

*Bug* - Pause button only freezes flights on the screen at the time of button click. Pending flights will still be added to map at regular interval. Any new markers added to the map during pause will begin movement simultaneously once resume is clicked.

Pending Flights Window

The pending flights button (found between the play and launch button) toggles the visibility of a list containing all pending flights that have not been launched yet. As the simaulation runs, flights will be removed from list after takeoff. Window is docked to right side of screen. Control bar buttons are still available when pending flights window is open. Window will populate with pending flights after scenario is loaded.

Load Scenario and Launch

The last two buttons in the bottom control bar are used to load a flight scenario and begin the simulation. The button with the arrow (last button in control bar) loads the scenario clicked. The pending flights button can be used to verify that a scenario has been loaded. The launch button begins the simulation, launching each flight at a regular interval.
